Example Usage

To send a message using the HTTP stream endpoint, you can use the following cURL command: 

      curl -X POST -H "Content-Type: application/json" -d '{{ "tool":
        "search_threads", "query": "from:hello@example.com" }}'
        http://localhost:3000{http_stream_route}

Or using the tools endpoint:

      curl "http://localhost:3000/tools/search_threads?query=from:hello@example.com"
